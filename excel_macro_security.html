The interactive Spreadsheet edition of the Field Guide is an Excel Spreadsheet application that uses macros. Your computer will consider these as a security threat. As such the spreadsheet needs to be "trusted" by Excel.

Normally Clicking on "Enable Content" will be all that is required, but it is possible to set up the "trust|" through the Excel Trust Center.

Below are the steps required to establish this "trust".

  1. Create a new folder such as Documents/MothsGuide
  2. Open Excel and navigate to File > Options > Trust Centre
  3. Click on the Trust Center Settings button and then "Trusted locations"
  4. Click on "Add new location", Browse to your new folder and click OK
  5. Right-click the link at the end of these instructions to download the macro-enabled spreadsheet.
  6. Move or copy the file from the Downloads folder to your new folder
  7. Open the spreadsheet from that folder.

If you receive a message about ActiveX then go to the "ActiveX Settings" section in the Trust Center. Select the "Prompt me before enabling all controls" option.
